Na linguagem JS o ; é opicional
Podemos usar "" ou '' só não podemos misturar "'
console.log //escrever no terminal
console.clear() // limpar a tela do terminal
Dica: "" Textos '' Importações

Nas estruturas de controle condicional e nos laços de 
repetições podemos omitir as chaves, porém só uma linha de
código é vinculada
Dica: Não omitir as chaves

const, var e let
